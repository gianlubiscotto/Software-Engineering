Detailed Description

template<typename R>
class DAG< R >

The class which represents a DAG.

A DAG is a container of Node and Leaf so a container for union of boxes (UBR).

DAG stores unions of boxes in n-dimensional space as n levels in which level 0 is always reserved for leaves.
