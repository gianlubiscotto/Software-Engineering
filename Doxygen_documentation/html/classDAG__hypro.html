Detailed Description

template<typename R>
class DAG_hypro< R >

The class which represents a DAG compatibile with hypro's types.

It it an inherited class from DAG but it is able to convert hypro::Box to BR. See class DAG for infos about what a DAG represents.

Method that determines if a hypro::Box is covered by a union of boxes inside the DAG-hypro.

First it performs the conversion from hypro::Box passed as parameter to BR and then it calls public method DAG::cover(const BR< R >& box, N i) const.
