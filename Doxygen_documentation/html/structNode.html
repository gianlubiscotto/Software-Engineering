Detailed Description

template<typename R>
struct Node< R >

The struct which represents a node inside the DAG.

A node is represented by a vector of points of discontinuity and links to other nodes/leaves to a lower level of the DAG.

A Node with its attributes represents a union of boxes (UBR).
